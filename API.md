## api

### Получить все события.
```
POST /api/event/getAll
```
Параметры: см. Parameters для получения всех событий

Ответ:
```js
{
    "status": "success",
    "events": [
        массив объектов <Событие>
    ]
}
```

### Получить одно событие.
```
POST /api/event/getOne
```
Параметры:
```js
        id: 1 // id события
```
Ответ:
```js
{
    "status": "success",
    "event": объект <Событие>
}
```
